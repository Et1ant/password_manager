# Password Manager (CLI)

Простой и функциональный менеджер паролей на Python для работы через терминал. Проект создан в рамках изучения основ работы с файловой системой и обработки данных.

## Возможности

* **Master Password**: Защита входа в программу (ограничение количества попыток).
* **CRUD операции**: Добавление, поиск, просмотр и удаление паролей.
* **File Storage**: Все данные сохраняются в локальный текстовый файл `passwords.txt`.
* **Security First**: Инструкция по безопасности и использованию `.gitignore`.

## Технологии

* **Language**: Python 3.x
* **OS**: Tested on Ubuntu Linux
* **Storage**: Plain text (.txt)

## Установка и запуск

1. Склонируйте репозиторий:
```bash
git clone https://github.com/ваш_логин/password-manager.git

```


2. Перейдите в папку проекта:
```bash
cd password-manager

```


3. Запустите программу:
```bash
python3 main.py

```



## Использование

При запуске программа потребует мастер-пароль (по умолчанию: `1111`). После успешного ввода откроется меню:

1. **Add**: Сохранить новый сервис, логин и пароль.
2. **Find**: Быстрый поиск по названию сервиса.
3. **View**: Список всех сохраненных учетных записей.
4. **Delete**: Удаление записи по названию сервиса.

---

### ⚠️ Важное примечание по безопасности

Этот проект является учебным. Пароли хранятся в открытом виде (plain text).
**Никогда не выкладывайте файл `passwords.txt` в публичный доступ!**

Для этого в репозитории должен быть файл `.gitignore` с одной строкой:

```text
passwords.txt

```
